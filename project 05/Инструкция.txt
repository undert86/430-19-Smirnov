	Портал сознательных граждан «Нарушениям.Нет» представляет собой информационную 
систему для помощи полиции по своевременной фиксации нарушений правил дорожного 
движения.
	Перед тем, как впервые воспользоваться услугами портала, гражданин должен 
зарегистрироваться. В ходе регистрации он указывает данные о себе: логин, пароль, фамилию, 
имя, отчество, телефон, адрес электронной почты.
	Войдя в систему, гражданин может сформировать обращение, указав регистрационный 
номер автомобиля и описание нарушения.
	Обращения граждан хранятся в системе. В каждом обращении указаны: регистрационный 
номер автомобиля, описание нарушения, статус обращения.

1. Установите OpenServer, скачав его с официального сайта - https://ospanel.io/download/.

2. Запустите OpenServer, на панели задач нажмите ПКМ на флажок и зайдите в Настройки -> Модули.
В модуле MySql поставить последнюю версию MySQL, с PHP проворачиваем такие же действия.

3. На панели задач также нажмите ПКМ на флажок -> Дополнительно -> PhpMyAdmin

Авторизуемся:
Пользователь - root, Пароля нет

Нажимаем Импорт -> Upload a file
Заходим в папку project5 databases, выбираем user.sql

4. Нажмите ПКМ на флажок -> Папка с проектами
Переносим туда папку project5

5. Чтобы запустить проект, нажмите ПКМ на флажок -> Мои проекты -> project5
